Download this project as a .zip file Download this project as a tar.gz file

What is this?

The Rymate Browser is a lightweight WebKit based browser written in C#. It is currently in developement, and when completed should have a few of these features:

  • Bookmarks
  • History
  • A New Tab Page
  • Webapps

Contributions are welcome at our GitHub repository.

Why Webkit?

Webkit is in my opinion one of the most compatible web browser engines out there. It is also one of the most popular, powering Safari, Chrome, the iOS Browser and the Android Browser. Its also much better than the default C# WebBrowser control in terms of web standards compliance.

How do I contribute to it?

All you need to do is checkout the git repository and start coding! You will need the Awesomium Web Browser Framework in order to compile it.

Preferred Coding Style

Comments and the default C# coding style please. I don't care about tabs/spaces, just make sure it all lines up.
